# Lifecycle Labels

Lifecycle labels accentuate a feature's status without derailing the user from their primary task. For details, see [the Primer documentation](https://primer.style/ui-patterns/feature-onboarding#lifecycles-labels).

## How to use

These labels can be used to annotate features in the UI without necessitating a disruptive dialog or popover.

When used in navigation components, they serve as the trailing visual and should not render a feedback link:

<Story of={LifecycleLabelsStories.InNavigation} />

Outside of navigation components, a feedback URL should always be provided:

<Story of={LifecycleLabelsStories.InPageContent} />
