# Simple ListView

<img alt="A Simple ListView component configured to show required properties. There are three overall parts to the component: the overall title, and then a list comprising of collections of list item subtitles, descriptions, and secondary actions. The secondary actions are a mix of select and text inputs and overflow action buttons." src={imageExample} />

The Simple ListView is a collection of items presented in a vertical list. Each item has:

* A [title](?path=/docs/recipes-simplelistview-documentation-accessibility-listitem--docs#listitem-title),
* An optional [title action](?path=/docs/recipes-simplelistview-documentation-accessibility-listitem--docs#listitem-title-action),
* An optional [description](?path=/docs/recipes-simplelistview-documentation-accessibility-listitem--docs#listitem-description), and
* One or more [secondary actions and information](?path=/docs/recipes-simplelistview-documentation-accessibility-listitem--docs#listitem-secondary-action-and-information)

### SimpleListView

A [flexible list wrapper](?path=/docs/recipes-simplelistview-documentation-accessibility-simple-listview--docs) used to contain and render collections of items.

### SimpleListItem

List [item wrapper](?path=/docs/recipes-simplelistview-documentation-accessibility-listitem--docs) that handles content layout, state, and interactions.
