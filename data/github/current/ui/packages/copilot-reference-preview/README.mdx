# CopilotReferencePreview

Displays a reference fed to or returned from Copilot chat.

## Example

Usage is simple: `<CopilotReferencePreview dismissable />`

## How to use

This component expects to be able to access `CopilotChatManagerContext` and `CopilotChatContext`. The reference to display will come from
`selectedReference` in `CopilotChatContext`.
