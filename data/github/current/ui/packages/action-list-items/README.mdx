# Action List Items

Primitives that uses ActionList under the hood and represents items within GitHub.

By creating these shared components this can be consumed by everyone creating React experiences including our ItemPicker. This will make sure those items throughout GitHub will look visually and audibly the same.

Tracking Issues:

* [https://github.com/github/platform-ux/issues/1363](https://github.com/github/platform-ux/issues/1363)
* [https://github.com/github/collaboration-workflows-flex/issues/378](https://github.com/github/collaboration-workflows-flex/issues/378)

![Examples of the actions list items (user, topic, repository, package type, and language) used in various settings such as multi select, single select, instant select, action.](https://github.com/primer/design/assets/980622/bf4a51be-316f-4bb5-af15-a25d6ca7d95f)

## How to use

Each component can be used in multiple scenarios and might look different depending on this use case. Let's take `ActionListItemUser` as an example.

### Action

![Example ActionListItemUser as an action use case with an profile pictire, username, and name](https://github.com/primer/design/assets/980622/d0da423d-ce0e-4af3-9511-e2c8b61e8b29)

#### Code example

```javascript
<ActionList> // No selectionVariant
  <ActionListItemUser selectType="action" avatarUrl="img.png" fullName="Manuel Mu単oz Solera" username="mamuso" />
</ActionList>
```

### Instant select

![Example ActionListItemUser as an instant use case with a check icon, profile pictire, username, and name](https://github.com/primer/design/assets/980622/6e3f0f01-4f93-45af-8d87-dfa028ba795a)

##### Code example

```javascript
<ActionList selectionVariant="single">
  <ActionListItemUser selectType="instant" avatarUrl="img.png" fullName="Manuel Mu単oz Solera" username="mamuso" />

```

### Multi select

![Example ActionListItemUser as an multi use case with a checkbox, profile pictire, username, and name](https://github.com/primer/design/assets/980622/65dae8ae-f597-4b9c-bcce-b1c45840e39d)

## Accessibility

The `role` is inferred by the container that wraps the ActionListItem variant (e.g. Select Panel, Action Menu) but make sure the role matches for your use case. If not, pass through the appropriate role into `role` prop
