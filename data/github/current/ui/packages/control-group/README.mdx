# ControlGroup

ControlGroup is a layout component for displaying controls. It should be used primarily on **parent-level** settings pages.

<Canvas of={DefaultStory.DefaultExample} />

***

### ControlGroup

The container component for `ControlGroup.Item` and `ControlGroup.LinkItem` components.

<ArgTypes of={ControlGroup} />

### ControlGroup.Item

The component that lays out `ControlGroup.Title` `ControlGroup.Description` and its control types.

Control types include:

* `ControlGroup.ToggleSwitch`
* `ControlGroup.Button`
* `ControlGroup.InlineEdit`
* `ControlGroup.Custom`
